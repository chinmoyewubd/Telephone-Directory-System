# 📞 Telephone Directory System (C Language)

A menu-driven Telephone Directory System implemented in C using arrays of structures and file handling.  
This project demonstrates fundamental programming concepts including structures, string manipulation, file system operations, and modular programming.

---

## 🚀 Project Overview

The Telephone Directory System allows users to:

- Load directory data from file
- Store directory data to file
- Insert new contact entries
- Delete existing contacts
- Edit contact information
- Search contacts by name (substring search)
- Display all contacts
- Sort contacts alphabetically

The program is fully menu-driven and runs in a console environment.

---

## 🛠 Technologies Used

- C Programming Language
- Standard C Libraries:
  - `stdio.h`
  - `string.h`
  - `stdlib.h`

## ⚙️ How to Run (Using CodeBlocks)

1. Open CodeBlocks
2. Create a new Console Application (C Language)
3. Add the following files:
   - main.c
   - directory.c
   - directory.h
4. Build the project (Ctrl + F9)
5. Run the project (Ctrl + F10)
